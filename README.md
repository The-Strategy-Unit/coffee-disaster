# Coffee disaster :coffee:💥

In the data science team we love coffee. We also seem to love accidentally pouring it all over our keyboards.

When a coffee disaster occurs, update the `coffee-disaster-tracking.csv` file with the disaster date in YYYY-MM-DD format and your GitHub username on the bottom row. 

The counter will automatically update on the next scheduled refresh

This dashboard tracks the number of days since our last coffee disaster. 

It also shows the number of coffee disasters each person has had since we started recording on 13th February 2026.

## Why does this exist?

This sidequest project is a useful place to test out GitHub actions, play with CSS styling, learn about accessibility or demo best practice GitHub features.
